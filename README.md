# Sales Performance Dashboard — Power BI

## Overview
Interactive 3-page Power BI dashboard built on
Superstore sales data to track business performance.

## Dataset
- Source: Kaggle — Superstore Sales
- Rows: 9,994 | Features: 21

## Tool
Power BI Desktop

## Dashboard Pages
1. Sales Performance — KPI cards, Category chart,
   Region donut, Year trend, 4 Slicers
2. Top 10 Sub Categories — Bar chart analysis
3. Sales by Segment — Segment and monthly trend

## Key Insights
- Technology leads with $836K in sales
- West region has highest sales at 31.6%
- Consumer segment contributes 50.5% of revenue
- 2017 was the highest sales year
